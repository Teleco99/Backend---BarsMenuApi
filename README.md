Esta API REST, desarrollada con Laravel, facilita la gestión de cartas y productos en bares. Los administradores pueden añadir, editar y eliminar menús y productos, con soporte para imágenes, descripciones detalladas y alérgenos. También incluye autenticación segura para garantizar que solo los administradores autorizados puedan realizar modificaciones en las cartas.

## Características principales:

- Gestión de menús y productos de forma sencilla.
- Autenticación de administradores para asegurar el acceso.
- Soporte para imágenes de productos y manejo de alérgenos.
- Almacenamiento eficiente de datos para un rendimiento óptimo.
